# AUTOMAÇÃO COM PYTHON

Alguns scripts simples que fui criando para estudar automação com Python.

## ATOMAÇÃO DE PLANILHA 

Este projeto foi desenvolvido em Python durante o curso Jornada Python da Hashtag Treinamentos, com o professor Lira.

O script consiste na automação do envio de dados de uma planilha de produtos para um site, utilizando as seguintes bibliotecas:

- pyautogui 
- pandas 
- time 

site utilizado para testes:
https://dlp.hashtagtreinamentos.com/python/intensivao/login

### Objetivo
- Automação de tarefas repetitivas
- Integração entre planilhas e sistemas web
- Controle de fluxo com Python
